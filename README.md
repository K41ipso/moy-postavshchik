# Мой.ПоставщИИк – Сервис поиска поставщиков продуктов питания

Интерактивный прототип сервиса, который помогает бизнесу быстро находить, сравнивать и выбирать поставщиков продуктов питания, собирая ключевую информацию для принятия решения.

**Демо (Vercel):** https://moy-postavshchik.vercel.app/

## Основная функциональность
* **Два режима работы:** "Умный подбор" (с алгоритмом скоринга соответствия) и "Каталог" (классический поиск).
* **Фильтрация и поиск:** По категории товара, названию, городу/региону.
* **Детальная информация:** Карточки и модальные окна содержат контакты, минимальный объем заказа, цены, условия доставки, наличие сертификатов и рейтинг.
* **Личные заметки:** Возможность добавить комментарий к поставщику (сохраняется локально).

## Расширенная функциональность (Bonus)
Для повышения практической применимости и удобства пользователя были реализованы следующие фичи:
1. **Трей "Избранное":** Мгновенное сохранение поставщиков в список с возможностью быстрого просмотра и удаления.
2. **Календарь закупок:** 
   - Визуальная сетка календаря с индикацией запланированных поставок.
   - Выбор диапазона дат для просмотра статистики.
   - Сводная аналитика: общий объем заказа (с разбивкой по единицам измерения) и итоговая сумма за выбранный период.
3. **Toast-уведомления:** Информирование пользователя о добавлении или удалении заказов.
4. **Полная адаптивность:** Корректное и удобное отображение на мобильных устройствах, планшетах и десктопах.

## Технологический стек и обоснование
* **Next.js (App Router) + React:** Обеспечивает высокую производительность, современный подход к компоновке компонентов и отличную DX (Developer Experience).
* **Tailwind CSS:** Позволяет быстро создавать адаптивный, чистый и "премиальный" UI с единой дизайн-системой.
* **LocalStorage:** Используется для сохранения состояния (избранное, заметки, заказы). *Обоснование:* Это позволяет продемонстрировать полную интерактивность, логику работы и UX прототипа без необходимости настройки бэкенда и базы данных проверяющим. Архитектура готова к бесшовной замене LocalStorage на реальные API-эндпоинты.

## Дополнительные детали
1. Данные поставщиков сгенерированы ИИ и носят исключительно демонстрационнный характер. Все совпадения случайны.
2. Данный демонстрационный сервис имеет огромный потенциал для расширения - от своей БД до подключения собственных ML или LLM моделей. Ограничением является лишь сервис, который предоставляет вычислительные мощности (сервис).
